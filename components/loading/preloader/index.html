Preloader

Preloader is a splash container that is used to hide the main contents until the page is fully loaded.

Reload this page and notice the black container above the main contents which is hidden once the pag is loaded.

#v-preloader.v-preloader

import * as Vevet from '../../../../../src/ts/index';

new Vevet.Application();

new Vevet.Preloader({
    container: '#v-preloader',
});
