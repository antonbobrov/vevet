PageLoad callbacks

Callbacks, that are triggered when the page is loaded.

Notice that the HTML and BODY elements may contain the class "v-loading" that is to be removed once the page is loaded.

Example

The page is 

import * as Vevet from '../../../../../src/ts/index';

// initialize the Vevet.Application instance
const app = new Vevet.Application();

// set onLoad callback
app.pageLoad.onLoaded(() => {
    setTimeout(() => {
        const indicatorEl = document.querySelector('#page-load-indicator');
        if (indicatorEl) {
            const badge = document.createElement('span');
            badge.classList.add('badge');
            badge.classList.add('bg-success');
            badge.innerHTML = 'loaded';
            indicatorEl.appendChild(badge);
        }
    }, 1500);
});
