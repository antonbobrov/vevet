Vevet Application

Vevet Application is the necessary instance of the whole library. It should be initialized before any component is used.

An example of initialization

import * as Vevet from '../../../src/ts/index';

export default new Vevet.Application({
    tablet: 1199,
    phone: 899,
    prefix: 'v-',
    viewportResizeTimeout: 100,
    easing: [0.25, 0.1, 0.25, 1],
});
