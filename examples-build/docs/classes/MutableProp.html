Class MutableProp<StaticProp, ChangeableProp>

A class for creating mutable properties that can change on window resize.

There are two ways to change properties:

  • To set a resize-listener on window (or use Viewport). When the window is resized, change the properties with the help of MutableProp.changeProp
  • The second way is to use the MutableProp and add responsive properties with help of MutableProp.addResponsiveProp.
