Class Page<StaticProp, ChangeableProp, CallbacksTypes>

Page is a class that may be used to initialize a page. The reason to use this class is that it allows to set a definite name for the page and manipulate with its states: creation, show, hide, and destruction. Notice that the states must be called a definite order: create -> show -> hide -> destroy and vice versa.
