## AutoHide {#autohide}

Scrollbars fade out when idle and reappear on scroll or drag.

Key prop: [`autoHide: true`](props#props.autoHide) on both vertical and horizontal instances.

## Min-size {#min-size}

Enforce a minimum thumb height even when content is very long.

Key prop: [`minSize: "90vh"`](props#props.minSize) — accepts CSS units (`px`, `rem`, `vh`, `svh`, etc.).

<VevetDemo id="NPxweQN" height={500} />
</section>

<section>
## No Auto-size {#no-auto-size}

Disable proportional thumb sizing.

Key prop: [`autoSize: false`](props#props.autoSize) — thumb keeps a fixed size regardless of content length.
