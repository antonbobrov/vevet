# Callbacks

<Head>
  <title>Scrollbar Callbacks | vevet.js</title>
</Head>

:::note
All **[Module callbacks](/docs/base/Module/#callbacks)** are available in this class.
:::

### `update`

Triggered when the scrollbar updates its position.

```ts
const instance = new Scrollbar({ }, {
  onUpdate: () => console.log('update'),
});
```

or:

```ts
const destruct = instance.on('update', () => console.log('update'));

// Cancel the callback
destruct();
```
