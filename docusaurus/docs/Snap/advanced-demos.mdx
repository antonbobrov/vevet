# Advanced Demos

<Head>
  <title>Snap Advanced Demos | vevet.js</title>
</Head>

These demos are **custom UI patterns** built on top of Snap â€” not built-in presets.

Most map `coord`, `progress`, and slide metadata in the `update` callback to CSS transforms. Some also use other vevet modules ([Timeline](/docs/Timeline/), [Swipe](/docs/Swipe/), [Responsive](/docs/base/Responsive/), and others). Expand the HTML/CSS/JS snippets below each demo to see the full implementation.

<section>
## Panorama

3D panorama carousel inspired by [Panorama Slider](https://panorama-slider.uiinitiative.com/).

Maps `progress` to `translateZ`, `rotateY`, and a horizontal offset in `onUpdate`. Uses `origin: "center"`, `freemode: true`, and `loop: true`.

<VevetDemo id="OPMzdaX" height={500} />
</section>

## Circular carousel

Slides positioned on a circle with `swipeAxis: "angle"`. Each slide's `progress` drives `sin` / `cos` placement and rotation.

<VevetDemo id="XJKQwyb" height={500} />
</section>

## Stack overlap

Overlapping stack: the active slide stays in place while previous slides shrink back in Z-space.

Driven by `progress` â€” `translateX`, `translateZ`, and `brightness` in `onUpdate`.

Uses `eventsEmitter: document.documentElement`, `freemode: true`, and `origin: "center"`.

## Fullsize parallax rotation

Vertical full-page carousel with child parallax via `data-snap-parallax-*` attributes on images and content.

See also [Snap parallax](/docs/Snap/parallax/). Snap moves slides with `translateY(${coord}px)`; parallax runs on nested elements.

</section>

Expo-style carousel inspired by [Expo Slider](https://expo-slider.uiinitiative.com/).

## Cards

Card stack with horizontal offset, Z-depth, and rotation from `progress`. Passed cards tuck beneath the active slide.

Uses `onSwipeStart` to track the gesture origin slide for asymmetric motion.

<VevetDemo id="QwNYJGo" height={500} />

<section>
## Expand slide {#expand-slide}

Freemode carousel where clicking a slide expands its width with [Timeline](/docs/Timeline/) and calls `slide.resize()` + `carousel.set()`.

Key props: `freemode: "sticky"`, `wheel: true`, `edgeFriction: 0.95`, `stickOnResize: false`.

<VevetDemo id="jEqdQZg" height={500} />
</section>

<section>
## Slick gap

Exponential gap between slides based on `progress` â€” each slide gets an extra offset: `coord + progress * absGap`.

Uses `freemode: "sticky"`, `origin: "center"`, and `shortSwipes: false`.

<VevetDemo id="myVpvoe" height={500} />
</section>
