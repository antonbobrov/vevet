# Callbacks

<Head>
  <title>InView Callbacks | vevet.js</title>
</Head>

:::note
All **[Module callbacks](/docs/base/Module/#callbacks)** are available in this class.
:::

### `in`

Fires when an element enters the viewport.

```ts
const observer = new InView({ }, {
  onIn: ({ element, direction }) => console.log(element, 'entered viewport', direction),
});
```

or:

```ts
const destruct = observer.on('in', ({ element, direction }) => {
  console.log(element, 'entered viewport', direction);
});

// Cancel the callback
destruct();
```

### `out`

Fires when an element leaves the viewport.

```ts
const observer = new InView({ }, {
  onOut: ({ element, direction }) => console.log(element, 'left viewport', direction),
});
```

or:

```ts
const destruct = observer.on('out', ({ element, direction }) => {
  console.log(element, 'left viewport', direction);
});

// Cancel the callback
destruct();
```
