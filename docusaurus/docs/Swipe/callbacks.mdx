# Callbacks

<Head>
  <title>Swipe Callbacks | vevet.js</title>
</Head>

:::note
All **[Module callbacks](/docs/base/Module/#callbacks)** are available in this class.
:::

### `start`

Swipe start event. See [ISwipeCoords](interfaces#iswipecoords).

```ts
const observer = new Swipe(
  {
    container: document.getElementById('container'),
  },
  {
    onStart: (coords) => console.log(coords),
  },
);
```

or:

```ts
const destruct = observer.on('start', (coords) => console.log(coords));

// Cancel the callback
destruct();
```
