# Swipe

Manages swipe interactions:

- Tracks pointer movement and detects direction
- Emits events on start, move, and end
- Two inertia modes: exponential (per-frame decay) and timeline (eased animation)
- Optional movement bounds with rubber-band overflow and bounce-back
- Calculates rotation
- Does not apply transforms — only provides computed coordinates
- Does not store swipe state after gesture ends (`diff` resets; use `total` for persistent offset)

## Initialization

:::caution

- `Swipe` does not transform elements — it only computes movement values.
- Per-gesture offsets live in `diff`; persistent applied movement is in `total` (see [accessors](./accessors#total)).
- Inertia is optional. [`inertiaThreshold`](./props#props.inertiathreshold): for `timeline` — linear speed only; for `exponential` — linear or angular.
- [`inertiaType`](./props#props.inertiatype) selects exponential (default) or timeline inertia; timeline uses [`inertiaDuration`](./props#props.inertiaduration) and [`inertiaEasing`](./props#props.inertiaeasing).
- [`bakeBounds`](./props#props.bakebounds) + [`overflow`](./props#props.overflow) enable rubber-band while dragging; exponential inertia can bounce inside baked bounds.
- Without inertia, [`releaseBounce`](./props#props.releasebounce) can snap `diff` back over [`bounceDuration`](./props#props.bounceduration).
- Timeline inertia clamps amplitudes to [`bakeBounds`](./props#props.bakebounds) by default, or uses [`velocityModifier`](./props#props.velocitymodifier) instead; exponential uses [`maxVelocity`](./props#props.maxvelocity).
  :::

```js
import { Swipe } from 'vevet';

const swipe = new Swipe(
  {
    container: document.getElementById('container'),
    grabCursor: true,
  },
  {
    onMove: ({ diff }, { container }) => {
      container.style.transform = `translate(${diff.x}px, ${diff.y}px)`;
    },
    onEnd: (data, { container }) => {
      container.style.transform = 'none';
    },
  },
);
```

Detect direction:

```js
swipe.on('toLeft', () => console.log('swipe to left'));
```

Add exponential inertia (default):

```js
import { Swipe } from 'vevet';

const swipe = new Swipe({
  container: document.getElementById('container'),
  inertia: true,
  inertiaType: 'exponential',
});
```

Timeline inertia (easing-based animation, optional amplitude clamp):

```js
import { EaseOutCubic } from 'easing-progress';
import { Swipe, clamp } from 'vevet';

const swipe = new Swipe({
  container: document.getElementById('container'),
  inertia: true,
  inertiaType: 'timeline',
  inertiaDuration: (distance) => clamp(distance, 500, 2000),
  inertiaEasing: EaseOutCubic,
  velocityModifier: (amp) => ({ ...amp, x: clamp(amp.x, -400, 400) }),
});
```

Constrain movement inside a parent and apply transforms from `total`:

```js
const thumb = document.getElementById('thumb');
const wrapper = document.getElementById('wrapper');

const swipe = new Swipe({
  container: thumb,
  inertia: true,
  bakeBounds: () => {
    const thumbBound = thumb.getBoundingClientRect();
    const wrapperBound = wrapper.getBoundingClientRect();

    return {
      x: [
        wrapperBound.left - thumbBound.left,
        wrapperBound.right - thumbBound.right,
      ],
      y: [
        wrapperBound.top - thumbBound.top,
        wrapperBound.bottom - thumbBound.bottom,
      ],
    };
  },
  onMove: ({ total }) => {
    thumb.style.transform = `translate(${total.x}px, ${total.y}px)`;
  },
});
```

Destroy all events:

```js
swipe.destroy();
```
