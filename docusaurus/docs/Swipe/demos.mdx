## Basic Demo {#basic-demo}

Overview of the main drag modes in one place: free movement, horizontal, vertical, and rotation.

Each instance uses pointer-space [`diff`](interfaces#iswipecoords) and resets the transform on release — no [`movement`](interfaces#iswipecoords) persistence.

Key props: `grabCursor: true`, `axis: "x"` / `axis: "y"`, `relative: true` (rotation).

## Multidirectional Swipe

Free drag on both axes without an axis lock.

Applies `diff.x` and `diff.y` in `onMove`; the element snaps back on `onEnd`.

<section>
## Slide Right

Horizontal slider thumb inside a track. Position persists across gestures via [`movement`](interfaces#iswipecoords).

Key props: [`thumb`](props#props.thumb), [`bounds`](props#props.bounds), [`inertia: true`](props#props.inertia), `grabCursor: true`.

Bounds are recalculated on resize with [`calculateBounds()`](methods#calculateBounds) and [`releaseBounce()`](methods#releaseBounce).

<VevetDemo id="NPdRrLy" height={500} />
</section>

## Slide Down

<section>
## Lever rotation {#lever-rotation}

Radial lever control: drag the handle (`thumb`) to rotate the dial. Angle is measured relative to the container center.

Key props: [`thumb`](props#props.thumb), [`relative: true`](props#props.relative), [`inertia: true`](props#props.inertia). Rotation accumulates via `step.angle`.

<VevetDemo id="bNpOmGo" height={500} />
</section>

<section>
## Detect Swipe Direction

Directional callbacks fire after the gesture ends once [`directionThreshold`](props#props.directionThreshold) is met.

Callbacks: `onStart`, `onToLeft`, `onToRight`, `onToTop`, `onToBottom`.

<VevetDemo id="ZYWVMwj" height={500} />
</section>

<section>
## Drag Inertia {#drag-inertia}

Release inertia without built-in bounds — limits are applied manually in `onMove`.

Key props: [`inertia: true`](props#props.inertia), `grabCursor: true`. Position is accumulated from `step` and clamped with `clamp()`.

<VevetDemo id="azNPaxx" height={500} />
</section>

<section>
## Drag Inertia Bounce

Movement-space drag inside a wrapper with automatic bounds and rubber-band overflow.

Key props: [`bounds`](props#props.bounds), [`overflow`](props#props.overflow), [`inertia: true`](props#props.inertia), `movement` for transforms.

<VevetDemo id="zxqbwVg" height={500} />

<section>
## Bouncing Ball

Custom physics on top of release inertia: the ball reverses direction when it hits the viewport edges.

Uses `hasInertia` in `onMove` to flip direction only during the inertia phase.

<VevetDemo id="pvyqOMV" height={500} />

## Vertical Scrolling

Simulated vertical scroll with CSS `translateY` instead of native overflow.

Key props: [`bounds`](props#props.bounds) on the y-axis, [`inertia: true`](props#props.inertia), `movement.y`, `grabCursor: true`.

<VevetDemo id="KwagYqE" height={500} />

<section>
## Fullpage Bounds

Drag an element anywhere on the page, constrained to the viewport center.

Key props: [`bounds`](props#props.bounds) relative to the page size, [`inertia: true`](props#props.inertia), `movement`. Recalculates bounds on resize.

<VevetDemo id="OPWRGBo" height={500} />
</section>

Caps how far release inertia can travel.

Key prop: [`inertiaDistanceModifier`](props#props.inertiaDistanceModifier) — clamps x/y distance to ±100 px.

<section>
## Image Viewer

Image viewer with pan, zoom, and pinch support. Combines Swipe with [`Pointers`](/docs/Pointers).

- Pan: [`movement`](interfaces#iswipecoords) + [`bounds`](props#props.bounds) that depend on [`scale`](accessors#scale)
- Zoom: mouse wheel, double-click, and two-finger pinch via [`setScale()`](methods#setScale)
- [`relative: true`](props#props.relative) for zoom-to-point

<VevetDemo id="rajMgoY" height={500} />
</section>
