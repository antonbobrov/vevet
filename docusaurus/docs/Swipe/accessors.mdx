# Accessors

<Head>
  <title>Swipe Accessors | vevet.js</title>
</Head>

:::note
All **[Module accessors](/docs/base/Module/#accessors)** are available in this class.
:::

### `accum`

Type: **[ISwipeVec3](interfaces#iswipevec3)**

Absolute path length since swipe start (`|step|` per axis).

```js
const instance = new Swipe();

instance.accum.x;
instance.accum.y;
instance.accum.angle;
```

### `container`

Type: `HTMLElement | SVGElement`

Event target element.

```js
const instance = new Swipe();

instance.container; // returns swipe container element
```

### `coords`

Type: **[ISwipeCoords](interfaces#iswipecoords)**

```js
const instance = new Swipe();

instance.coords; // returns all coordinates data
```

### `current`

Type: **[ISwipeState](interfaces#iswipestate)**

Current coordinate.

```js
const instance = new Swipe();

instance.current.x;
instance.current.y;
instance.current.angle;
```

### `diff`

Type: **[ISwipeState](interfaces#iswipestate)**

Difference between start and current coordinates.

```js
const instance = new Swipe();

instance.diff.x;
instance.diff.y;
instance.diff.angle;
```

### `hasBounce`

Type: `boolean`

Indicates if release bounce-back animation is active ([`releaseBounce`](/docs/Swipe/props#props.releasebounce), `diff` out of [`bakeBounds`](/docs/Swipe/props#props.bakebounds)).

```js
const instance = new Swipe();

instance.hasBounce; // true or false
```

### `hasInertia`

Type: `boolean`

Indicates if inertia is active.

```js
const instance = new Swipe();

instance.hasInertia; // true or false
```

### `isSwiping`

Type: `boolean`

Indicates if a swipe is active.

```js
const instance = new Swipe();

instance.isSwiping; // true or false
```

### `prev`

Type: **[ISwipeState](interfaces#iswipestate)**

Previous coordinate.

```js
const instance = new Swipe();

instance.prev.x;
instance.prev.y;
instance.prev.angle;
```

### `start`

Type: **[ISwipeState](interfaces#iswipestate)**

Start coordinate.

```js
const instance = new Swipe();

instance.start.x;
instance.start.y;
instance.start.angle;
```

### `step`

Type: **[ISwipeState](interfaces#iswipestate)**

Difference between current and previous coordinates.

```js
const instance = new Swipe();

instance.step.x;
instance.step.y;
instance.step.angle;
```

### `total`

Type: **[ISwipeVec3](interfaces#iswipevec3)**

Applied movement since instance initialization (after rubber-band). Use this for persistent transforms.

```js
const instance = new Swipe();

instance.total.x;
instance.total.y;
instance.total.angle;
```
