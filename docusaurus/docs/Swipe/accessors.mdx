# Accessors

<Head>
  <title>Swipe Accessors | vevet.js</title>
</Head>

:::note
All **[Module accessors](/docs/base/Module/#accessors)** are available in this class.
:::

### `coords`

Type: **[ISwipeCoords](interfaces#iswipecoords)**

Full snapshot: pointer fields plus movement-space fields (`movement`, `prevMovement`, `scale`).

```js
const instance = new Swipe();

instance.coords.movement.x;
instance.coords.scale;
instance.coords.diff.x;
```

### `container`

Type: `HTMLElement | SVGElement`

Coordinate reference element (`props.container`).

```js
const instance = new Swipe();

instance.container;
```

### `movement`

Type: **[ISwipeVec3](interfaces#iswipevec3)**

Total displacement in **movement space** (rubber + snap). Use for transforms when `bounds` / `snap` are enabled.

```js
const instance = new Swipe({
  bounds: () => ({ x: [0, 200] }),
  onMove: ({ movement, scale }) => {
    el.style.transform = `translate(${movement.x}px, ${movement.y}px) scale(${scale})`;
  },
});

instance.movement.x;
```

### `scale`

Type: `number`

Current scale modifier in movement space. Default is `1`. Updated by [`setScale`](methods#setscale). Included in [`ISwipeCoords`](interfaces#iswipecoords) on callbacks.

```js
const instance = new Swipe({
  onMove: ({ movement, scale }) => {
    el.style.transform = `translate(${movement.x}px, ${movement.y}px) scale(${scale})`;
  },
});

instance.scale;
```

### `start`

Type: **[ISwipeState](interfaces#iswipestate)**

Pointer position at swipe start.

```js
instance.start.x;
instance.start.time;
```

### `prev`

Type: **[ISwipeState](interfaces#iswipestate)**

Previous pointer position.

```js
instance.prev.x;
```

### `current`

Type: **[ISwipeState](interfaces#iswipestate)**

Current pointer position.

```js
instance.current.x;
instance.current.angle;
```

### `diff`

Type: **[ISwipeState](interfaces#iswipestate)**

Offset from swipe start to current pointer position (pointer space).

```js
instance.diff.x;
instance.diff.angle;
```

### `step`

Type: **[ISwipeState](interfaces#iswipestate)**

Offset from previous to current pointer position.

```js
instance.step.x;
```

### `accum`

Type: **[ISwipeVec3](interfaces#iswipevec3)**

Absolute path length since swipe start (`|step|` per axis).

```js
instance.accum.x;
instance.accum.angle;
```

### `hasInertia`

Type: `boolean`

Whether release inertia (RAF) is running.

```js
instance.hasInertia;
```

### `hasBounce`

Type: `boolean`

Whether overflow bounce-back timeline is running.

```js
instance.hasBounce;
```

### `isSwiping`

Type: `boolean`

Whether a swipe gesture is in progress.

```js
instance.isSwiping;
```
