## Static Props

Static properties are set during initialization and cannot be modified later.

### `container` {#props.container}

- **Type:** `HTMLElement | SVGElement`
- Event listener container.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
});
```

### `thumb` {#props.thumb}

- **Type:** `HTMLElement | SVGElement | null`
- **Default:** `null`
- An element that triggers swipe start.  
  Movement is still calculated relative to the container — `thumb` only defines where a user can grab.
- **See [demo](demos#lever-rotation)**

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  thumb: document.getElementById('thumb'),
});
```

### `buttons` {#props.buttons}

- **Type:** `number[]`
- **Default:** `[0]`
- Determines which mouse buttons trigger events.
  - 0: Main button pressed, usually the left button or the un-initialized state
  - 1: Auxiliary button pressed, usually the wheel button or the middle button (if present)
  - 2: Secondary button pressed, usually the right button
  - 3: Fourth button, typically the Browser Back button
  - 4: Fifth button, typically the Browser Forward button
- See [MouseEvent.button](https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent/button).
- **See [demo](demos#right-click-drag)**

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  buttons: [0], // left button click
  buttons: [2], // right button click
  buttons: [0, 2], // both buttons click
});
```

### `pointers` {#props.pointers}

- **Type:** `number`
- **Default:** `1`
- Required pointer count to activate swiping.  
  Useful for touch gestures such as two-finger.  
  If fewer pointers are present, swipe will not start.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  pointers: 2, // means that swipe will start only with two fingers
});
```

### `disableUserSelect` {#props.disableuserselect}

- **Type:** `boolean`
- **Default:** `true`
- Disables text selection while dragging (passed to the internal `Pointers` instance).

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  disableUserSelect: true,
});
```

## Mutable Props

Mutable properties can be updated at runtime using `.updateProps()`.

:::tip Adaptive props
To apply different values by viewport or device, use **[Responsive](/docs/base/Responsive/#example-with-module)** with this component as the source. Props will update automatically when breakpoints change.
:::

### `enabled` {#props.enabled}

- **Type:** `boolean`
- **Default:** `true`
- Enables or disables swipe events.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  enabled: false,
});

// change value
swipe.updateProps({
  enabled: true,
});
```

### `relative` {#props.relative}

- **Type:** `boolean`
- **Default:** `false`
- Calculates coordinates relative to the container rather than the page.
- **See [demo](demos#rotation-swipe)**

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  relative: true,
});

// change value
swipe.updateProps({
  relative: false,
});
```

### `axis` {#props.axis}

- **Type:** `null | 'x' | 'y'`
- **Default:** `null`
- Primary swiping axis.
- **See [demo](demos#basic-demo)**

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  axis: 'x',
});

// change value
swipe.updateProps({
  axis: 'y',
});
```

### `ratio` {#props.ratio}

- **Type:** `number`
- **Default:** `1`
- Swipe move ratio (speed multiplier).

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  ratio: 1,
});

// change value
swipe.updateProps({
  ratio: 2,
});
```

### `grabCursor` {#props.grabCursor}

- **Type:** `boolean`
- **Default:** `false`
- Shows "grab" and "grabbing" cursors during interaction.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  grabCursor: true,
});

// change value
swipe.updateProps({
  grabCursor: false,
});
```

### `willAbort` {#props.willAbort}

- **Type:** `(props: ISwipeCanMoveArg) => boolean`
- **Default:** `() => false`
- Runs before swipe starts and determines whether the gesture should be blocked.  
  Allows disabling swipe in specific UI states or when gesture conditions are not met.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  willAbort: () => false,
});

// change value
swipe.updateProps({
  willAbort: ({ diff }) => diff.x > 5, // will prevent too fast swipe
});
```

### `threshold` {#props.threshold}

- **Type:** `number`
- **Default:** `5`
- Minimum swipe distance (px) to trigger swipe start.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  threshold: 5,
});

// change value
swipe.updateProps({
  threshold: 3,
});
```

### `minTime` {#props.minTime}

- **Type:** `number`
- **Default:** `0`
- Minimum duration (ms) to trigger swipe move.
- Means that user click the container, waits some time, and only then swipe is available.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  minTime: 0,
});

// change value
swipe.updateProps({
  minTime: 0,
});
```

### `directionThreshold` {#props.directionThreshold}

- **Type:** `number`
- **Default:** `50`
- Minimum swipe distance (px) for directional callbacks.
- Swipe supports direction detection (toTop, toBottom, toLeft, toRight).
- To trigger a directional callback a user must swipe a certain amount of pixels - `directionThreshold`.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  directionThreshold: 50,
});

// change value
swipe.updateProps({
  directionThreshold: 10,
});
```

### `preventEdgeSwipe` {#props.preventEdgeSwipe}

- **Type:** `boolean`
- **Default:** `true`
- Prevents edge swiping (iOS swipe-back gesture).

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  preventEdgeSwipe: true,
});

// change value
swipe.updateProps({
  preventEdgeSwipe: false,
});
```

### `edgeSwipeThreshold` {#props.edgeSwipeThreshold}

- **Type:** `number`
- **Default:** `20`
- Edge swipe threshold (px) from the left/right edge.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  edgeSwipeThreshold: 20,
});

// change value
swipe.updateProps({
  edgeSwipeThreshold: 25,
});
```

### `preventTouchMove` {#props.preventTouchMove}

- **Type:** `boolean`
- **Default:** `true`
- Prevents `touchmove` from scrolling the page while swiping.  
  Recommended to keep enabled for horizontal/vertical drags inside scrollable layouts.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  preventTouchMove: true,
});

// change value
swipe.updateProps({
  preventTouchMove: false,
});
```

### `requireCtrlKey` {#props.requireCtrlKey}

- **Type:** `boolean`
- **Default:** `false`
- Requires Ctrl key for swipe (mouse only).

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  requireCtrlKey: true,
});

// change value
swipe.updateProps({
  requireCtrlKey: false,
});
```

### `overflow` {#props.overflow}

- **Type:** `() => number`
- **Default:** `() => 50`
- Rubber-band overflow past [`bakeBounds`](#props.bakebounds) during drag and exponential inertia bounce (px or degrees, same units as the axis).

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  overflow: () => 50,
});

// change value
swipe.updateProps({
  overflow: () => 20,
});
```

### `bakeBounds` {#props.bakebounds}

- **Type:** `null | (() => ISwipeBounds | null)`
- **Default:** `null`
- Returns `[min, max]` per axis for `diff` (offset from swipe start). Cached on swipe start; re-fetched on inertia start.
- **Drag:** rubber-band via [`overflow`](#props.overflow).
- **Exponential inertia:** per-frame bounce inside bounds.
- **Timeline inertia:** amplitudes are clamped to baked bounds when [`velocityModifier`](#props.velocitymodifier) is not set.
- **See [index example](/docs/Swipe/#initialization)**

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  bakeBounds: () => ({
    x: [-100, 100],
    y: [-50, 50],
    angle: [-45, 45],
  }),
});
```

### `releaseBounce` {#props.releasebounce}

- **Type:** `() => boolean`
- **Default:** `() => true`
- When inertia does not run (or fails), enables bounce-back animation if `diff` is outside [`bakeBounds`](#props.bakebounds).  
  Return `false` to skip snap-back (e.g. during another transition).

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  releaseBounce: () => true,
});

// change value
swipe.updateProps({
  releaseBounce: () => false,
});
```

### `bounceDuration` {#props.bounceDuration}

- **Type:** `number`
- **Default:** `250`
- Duration (ms) of the release bounce-back timeline when [`releaseBounce`](#props.releasebounce) allows it and `diff` is out of bounds.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  bounceDuration: 250,
});

// change value
swipe.updateProps({
  bounceDuration: 400,
});
```

### `inertia` {#props.inertia}

- **Type:** `boolean`
- **Default:** `false`
- Enables inertia after pointer release. Mode is controlled by [`inertiaType`](#props.inertiatype).
- **See [demo](demos#drag-inertia)**

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertia: true,
});

// change value
swipe.updateProps({
  inertia: false,
});
```

### `inertiaType` {#props.inertiatype}

- **Type:** `'exponential' | 'timeline'`
- **Default:** `'exponential'`
- Inertia animation mode:
  - **`exponential`** — per-frame velocity decay; optional [`bakeBounds`](#props.bakebounds) bounce ([`inertiaDecay`](#props.inertiadecay), [`inertiaBounceEase`](#props.inertiabounceease), [`maxVelocity`](#props.maxvelocity))
  - **`timeline`** — eased timeline from release speed ([`inertiaDuration`](#props.inertiaduration), [`inertiaEasing`](#props.inertiaeasing)); [`bakeBounds`](#props.bakebounds) or [`velocityModifier`](#props.velocitymodifier) limit displacement
- **See [demo](demos#drag-inertia-bounce)** (timeline)

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertia: true,
  inertiaType: 'exponential',
});

// change value
swipe.updateProps({
  inertiaType: 'timeline',
});
```

### `inertiaRatio` {#props.inertiaratio}

- **Type:** `number`
- **Default:** `1`
- Inertia strength multiplier applied to release velocity (both modes).

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertiaRatio: 1,
});

// change value
swipe.updateProps({
  inertiaRatio: 2,
});
```

### `inertiaThreshold` {#props.inertiathreshold}

- **Type:** `number`
- **Default:** `1`
- Minimum pointer speed to start inertia (px/s for linear, deg/s for angular).
- **`timeline`:** linear speed only.
- **`exponential`:** linear **or** angular speed must meet the threshold.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertiaThreshold: 1,
});

// change value
swipe.updateProps({
  inertiaThreshold: 5,
});
```

### `inertiaDecay` {#props.inertiaDecay}

- **Type:** `number`
- **Default:** `0.025`
- Inertia decay per frame. Higher values stop inertia sooner.  
  Applicable for exponential-based inertia only.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertiaDecay: 0.025,
});

// change value
swipe.updateProps({
  inertiaDecay: 0.05,
});
```

### `inertiaBounceEase` {#props.inertiaBounceEase}

- **Type:** `number`
- **Default:** `0.1`
- Easing factor for inertia bounce back into bounds (per frame, FPS-independent).  
  Applicable for exponential-based inertia only.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertiaBounceEase: 0.1,
});

// change value
swipe.updateProps({
  inertiaBounceEase: 0.2,
});
```

### `velocityModifier` {#props.velocitymodifier}

- **Type:** `false | ((amplitude: ISwipeVec3) => ISwipeVec3)`
- **Default:** `false`
- Modifies timeline inertia **displacement amplitudes** (px for `x`/`y`, degrees for `angle`) after release speed is converted to px/s / deg/s.  
  When set, replaces the default [`bakeBounds`](#props.bakebounds) clamp on timeline release.  
  Use for custom limits (e.g. carousel track). Applicable when [`inertiaType`](#props.inertiatype) is `'timeline'` only.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertiaType: 'timeline',
  velocityModifier: (amp) => ({
    ...amp,
    x: clamp(amp.x, -slideMin, slideMax),
  }),
});
```

### `maxVelocity` {#props.maxVelocity}

- **Type:** **[ISwipeVec3](interfaces#iswipevec3)**
- **Default:** `{ x: 7, y: 7, angle: 3 }`
- Max release velocity per axis (coordinate units / ms for `x`/`y`, degrees / ms for `angle`) applied on release before inertia starts.  
  A falsy axis value clamps that axis to `0` (no inertia on the axis).

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  maxVelocity: { x: 7, y: 7, angle: 3 },
});

// change value
swipe.updateProps({
  maxVelocity: { x: 10, y: 10, angle: 0 },
});
```

### `inertiaDuration` {#props.inertiaDuration}

- **Type:** `(distance: number) => number`
- **Default:** `(distance) => clamp(distance, 500, 2000)`
- Inertia animation duration (ms). `distance` is linear release speed in px/s.  
  Applicable when [`inertiaType`](#props.inertiatype) is `'timeline'`.

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertiaDuration: (distance) => clamp(distance, 500, 2000),
});

// change value
swipe.updateProps({
  inertiaDuration: 500,
});
```

### `inertiaEasing` {#props.inertiaEasing}

- **Type:** `TEasingType`
- **Default:** `EaseOutCubic`
- Easing function for timeline inertia progress (`0` → `1`).
- See **[easing types](/docs/utils#easing-types)** for more information.  
  Applicable when [`inertiaType`](#props.inertiatype) is `'timeline'`.
- **See [demo](demos#drag-inertia-bounce)**

```js
const swipe = new Swipe({
  container: document.getElementById('container'),
  inertiaEasing: EaseOutCubic,
});

// change value
swipe.updateProps({
  inertiaEasing: EaseOutBack,
});
```
